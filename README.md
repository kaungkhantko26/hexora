# Hexora 🎵

Hexora is a lightweight lyrics website focused on making songs easier to **discover, organize, and read in one place**.

**Live demo:** https://hexora.kaungkhantko.top/

---

## What you can do

- **Find Song Lyrics** by searching with:
  - artist name
  - song name
  - or both
- Browse an **Artist Profile List**
- View a library of **Song Names**
- Use **Toggle Theme** (light/dark)

---

## Pages

- `/` — Find Song Lyrics (search)
- `/artist-profiles/` — Artist Profile List
- `/song-names/` — Song Names library
- `/about/` — About Hexora

---

## Project goal

Build a clean and simple space where people can search songs, explore artists, and quickly open song lyrics.
